Project Name: Cara - A messenger-based chatbot 
Rolls: 1908044-1908049

Procedures in brief:

1. At first, created an app on our main folder. 
2. Then connected the app Cara with the main local host. Then added all the codes in views.py. 
3. Then for a local server, we used ngrok to do that. 
4. After creating our own server we created a token by FB developer account. 
5. After that, we added the final code of the chatbot

Platforms used: 
1. VS code for coding
2. Django as a local host
3. Ngrok as the public server


NB. There is a JSON file in the Github repository which can be modified according to the user's requirements
